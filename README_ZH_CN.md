# 什么是Litewars?
## 一个我的世界小游戏插件
你需要收集资源来对抗你的对手，同时保护自己的床不被破坏，如果你的床被破坏，你就无法再次重生
## 如何使用这个插件?
只需要将此插件放入您的服务器根目录下的plugins文件夹下(在关闭服务器的时候)，然后启动服务器让他生成所需的配置文件，然后修改config.yml来自定义您对此插件的设置
